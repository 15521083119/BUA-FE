#### repo

提供两个方法：

1. getPaginationSKU
2. getDetailSKU

`getPaginationSKU()` 主要是对数据进行分页查询，支持前端分页无限滚动商品列表业务。该方法接收一个对象，该对象需要提供如下两个属性：

limit ：每次分页的数据量

start：分页数据的起始点

`getDetailSKU()`主要提供商品的详细信息，为商品详细展示页面提供数据。该方法接受一个`sku_id`，用来指示如何在数据库中进行查询。
