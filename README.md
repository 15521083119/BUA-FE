### 闭环思维



>  并不是之后想去做全栈，或者认为前端不好玩没前景。我只是认为作为一名软件工程师，有余力的情况下是需要对整个研发流程有一定理解和体会的。这样在多人协作时才能发挥出最大价值，能够换位思考去理解他人的难处，而不是互相甩锅最后事情没有做好。

因此在设计整套毕设的时候就考虑到我不想做一个`toy project`,在这个过程中主要参考并学习如下视频课：

- java全栈

- 上线部署
